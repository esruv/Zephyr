# Zephyr

Simple & lightweight World of Warcraft addon that customizes the default UI.

## Features (W.I.P)

-   Class colored health bars.
-   Hide PvP icon.
-   Hide feedback text on player/pet frame.
-   Hide reputation color on target/focus frame.
-   Combat Indicator for target/focus frame.

## Options

You can use **`/zephyr`** or **`/zp`** to open the options menu.
